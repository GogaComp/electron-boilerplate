# Electron boileplate

Electron boileplate with webpack (babel, sass).

## Installation

```javascript
$ npm i
```

or

```javascript
$ yarn
```

## Usage

### Start electron:

```javascript
$ npm start
```

or

```javascript
$ yarn start
```

### Build webpack:

```javascript
$ npm run build
```

or

```javascript
$ yarn build
```

### Generate executable build:

```javascript
$ npm run dist
```

or

```javascript
$ yarn dist
```
