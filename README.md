# Electron boileplate

Electron boileplate with webpack (babel, sass).

## Installation

```javascript
$ npm i
```

or

```javascript
$ yarn
```

## Usage

### Start electron:

```javascript
$ yarn start
```

### Build webpack:

```javascript
$ yarn build
```

### Generate executable build:

```javascript
$ yarn dist
```

### Create executable file (package):

#### Windows:

```javascript
$ yarn package-win
```

#### Linux:

```javascript
$ yarn package-linux
```
